# documentation

Specs, ADRs, design docs, project-level prose (README, MILESTONES, VISION, CHANGELOG, CONTRIBUTING, CLAUDE.md), this holistic spec at `docs/architecture/`, and the claude-ultra apparatus under `.changes/`.

This module is broad by design: it captures every artifact that documents intent or records process state. Architectural changes nearly always touch this module (at minimum, decisions logs and proposed amendments).

**Paths:** see [`.changes/catalogs/modules.toml`](../../../.changes/catalogs/modules.toml).
