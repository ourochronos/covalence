# extensions

Extension manifests — declarative YAML packages that add entity types, relationship types, domains, alignment rules, services, and lifecycle hooks without modifying the engine. Five default extensions ship: core, code-analysis, spec-design, research, agent-memory.

**Paths:** see [`.changes/catalogs/modules.toml`](../../../.changes/catalogs/modules.toml).
